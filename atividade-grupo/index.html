Reset, Reboot e Normalize

Reset CSS

O que é: Remove todos os estilos padrão do navegador (margens, paddings, tipografia, etc.), deixando a página "crua".

Como se aplica: Adicionando um arquivo reset.css que zera os estilos.

Vantagens: Maior controle total sobre cada detalhe do design.

Reboot CSS

O que é: Uma evolução do reset criada pelo Bootstrap. Redefine estilos de forma equilibrada, já aplicando melhorias modernas.

Como se aplica: Incluindo o reboot.css do Bootstrap ou usando frameworks similares.

Vantagens: Corrige inconsistências entre navegadores e mantém uma base tipográfica acessível.

Normalize CSS

O que é: Não zera os estilos, mas padroniza os elementos para ficarem consistentes entre diferentes navegadores.

Como se aplica: Importando normalize.css no início do projeto.

Vantagens: Mantém estilos úteis e garante compatibilidade sem remover tudo.
